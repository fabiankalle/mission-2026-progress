# Mission 2026 Progress

Kostenlose iPhone-Web-App (PWA) für Daily Check-ins, Gewichts- und Diätverlauf, Cardio-Wochenziel, Creatin und detailliertes Trainingstracking. Alle neuen persönlichen Daten bleiben lokal auf dem Gerät.

## Version 3.0.1

- Daily Check-in für frei wählbare Tage; Felder können später ergänzt, geändert oder gelöscht werden
- Gewichtsziel mit Reststrecke und Fortschrittsanzeige
- Kalorienziel-Historie mit Gültigkeitsdatum
- Wochenübersichten für Kalorien, Protein, Schlaf, Schritte und Creatin
- Cardio-Wochenziel frei einstellbar, wahlweise Montag–Freitag oder Montag–Sonntag
- direkte Anzeige der noch fehlenden Cardio-Minuten
- korrekte Kalenderwochen, echte Datumsabstände und 7-Kalendertage-Gewichtstrend
- Plateau-Frühwarnung nach einer Woche und bestätigte Plateau-Meldung nach zwei vollständigen Wochen
- durchschnittliche wöchentliche Gewichtsabnahme und prozentualer Gewichtsverlust
- Trainingsplan Gym A, Zuhause und Gym B
- Dropsätze korrekt als eine Serie mit Startgewicht plus zwei Reduzierungen
- vorherige Trainingswerte sichtbar; Übernahme nur per Button
- nur abgehakte Sätze werden gespeichert
- Übungen überspringen, Einheiten bearbeiten und löschen
- getrennte Rechts-/Links-Auswertung bei einarmigen Übungen
- GORNATION-Bänder: Ultra Light 2–7 kg, Light 5–15 kg, Medium 10–30 kg, Heavy 20–40 kg
- automatische Trainingsentwürfe, Pausentimer und optionale Trainingsdauer
- Bestleistungen, geschätztes 1RM, Wiederholungsrekorde, Volumen und Trainingsplateau-Analyse
- Diagrammzeiträume 4/8/12 Wochen oder alles; genaue Werte per Antippen
- JSON-Backup, CSV-Export und Backup-Erinnerung
- Netzwerk-zuerst-Update für die App-Seite, damit neue GitHub-Versionen schneller erscheinen

## GitHub aktualisieren

1. ZIP entpacken.
2. Im bestehenden Repository **Add file → Upload files** öffnen.
3. Mindestens `index.html`, `sw.js` und `README.md` hochladen; vorhandene Dateien ersetzen.
4. **Commit changes** drücken.
5. Unter **Actions** auf den grünen Haken des Deployments warten.
6. Die App einmal in Safari öffnen und neu laden. Bei sichtbarer Update-Meldung auf **Aktualisieren** tippen.
7. Die Home-Screen-App komplett schließen und erneut öffnen.

Die GitHub-Pages-Einstellungen müssen nicht erneut geändert werden.
